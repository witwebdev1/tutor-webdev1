##Nesting Lists

It is possible to nest lists to achieve an outline type of display. To do this the innermost list must be enclosed within the outermost list.

HTML Code
~~~
<ol>
   <li>topic 1
   <ol>
     <li>subtopic A</li>
     <li>subtopic B
     <ol>
       <li>sub-subtopic i</li>
       <li>sub-subtopic ii</li>
     </ol>
     </li>
     <li>subtopic C</li>
     </ol></li>
   <li>topic 2</li>
   <li>topic 3</li>
</ol>
~~~
